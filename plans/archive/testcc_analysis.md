# TestCC.cs 구조 분석 및 기능 명세

이 문서는 `/plan` 워크플로우에 의해 자동 생성되었습니다. `TestCC` 클래스의 현재 구조와 구현된 기능을 분석합니다.

## 1. 개요
`TestCC`는 `CharacterController`를 사용하여 플레이어의 이동, 점프 및 물리 반응을 처리하는 컴포넌트입니다.

## 2. 주요 변수 (Fields)
- `moveSpeed`: 기본 이동 속도 (5.0)
- `jumpForce`: 점프 시 가해지는 수직 힘 (5.0)
- `maxJumps`: 최대 허용 점프 횟수 (2회)
- `_airVelocity`: 공중에서 유지되는 수평 관성 속도
- `_startPosition`: 추락 시 복귀할 최초 시작 위치

## 3. 구현된 핵심 기능
### 3.1. 기본 이동 및 시점 회전
- 마우스 X 입력을 통해 캐릭터의 Yaw(좌우) 회전을 처리합니다.
- WASD 입력을 기반으로 수평 이동을 수행합니다. 지면에 있을 때만 새로운 이동 입력을 수용합니다.

### 3.2. 2단 점프 (Double Jump)
- `cc.isGrounded` 상태일 때 점프 횟수를 초기화합니다.
- 공중에서도 남은 점프 횟수가 있다면 추가 점프를 수행할 수 있습니다.

### 3.3. 공중 속도 유지 (Air Momentum)
- 지면을 떠나는 순간의 `move` 벡터를 `_airVelocity`에 저장합니다.
- 공중(`!isGrounded`)에서는 방향키 입력을 무시하고 저장된 속도를 유지하여 자연스러운 관성을 구현합니다.

### 3.4. 추락 방지 리스폰 (Fall Respawn)
- 매 프레임 위치를 체크하여, 시작 위치 대비 10m 아래로 추락할 경우 `_startPosition`으로 즉시 이동시킵니다.

## 4. 코드 흐름 분석
1. `Start()`: `CharacterController` 캐싱 및 시작 위치 저장.
2. `Update()`:
   - 마우스 회전 적용.
   - 지면 여부에 따른 이동 속도(`move`) 결정.
   - 점프 입력 처리 및 수직 속도 계산.
   - 중력 적용 및 최종 이동(`cc.Move`).
   - 추락 감지 및 리스폰 처리.
