# Animation Editor 사용설명서

IronRose 엔진의 애니메이션 에디터. 키프레임 기반 프로퍼티 애니메이션을 생성·편집한다.

---

## 기본 구조

```
┌─────────────────────────────────────────────────────────┐
│  메뉴바 (File / Edit)                                    │
├─────────────────────────────────────────────────────────┤
│  툴바: Target | ● Rec | Play/Pause/Stop | |< >| | +Key  │
│        Time | Speed | FR | WrapMode | Length | Save      │
│        [Dope] [Curves]                                   │
├──────────────┬──────────────────────────────────────────┤
│  트랙 리스트   │  타임라인 (Dopesheet / Curves)            │
│  + Properties │  ┌─ 룰러 (시간축 + 이벤트 마커) ─────┐   │
│               │  │                                    │   │
│  ▾ Transform  │  │  ◆ ◆    ◆       ◆  (Dopesheet)   │   │
│    position.x │  │  ── curve ──     (Curves)          │   │
│    position.y │  │                                    │   │
│    rotation.z │  └────────────────────────────────────┘   │
├──────────────┴──────────────────────────────────────────┤
│  (키프레임/이벤트 선택 시 Inspector 표시)                  │
└─────────────────────────────────────────────────────────┘
```

## 툴바

### Target Animator

드롭다운으로 씬에서 Animator 컴포넌트가 있는 오브젝트를 선택한다. 대부분의 기능(재생, 녹화, 프리뷰)은 타겟 선택이 필요하다.

### Record 모드 (● Rec / ● Stop)

녹화를 켜면:
- Inspector에서 Transform(Position, Rotation, Scale) 값을 변경하거나
- 씬 뷰에서 기즈모로 오브젝트를 이동/회전/스케일하거나
- Inspector에서 컴포넌트 프로퍼티(float, int, bool, Vector3, Color)를 변경하면

현재 playhead 위치에 자동으로 키프레임이 생성된다.

- 타이틀바에 `[REC]` 표시
- 커브가 없으면 자동 생성
- Undo 지원 (기즈모 드래그 = 1회 undo, Inspector 변경 = 즉시 undo)
- PlayMode 진입 시 자동 해제

### 재생 컨트롤

| 버튼 | 기능 |
|------|------|
| Play | 애니메이션 재생 시작 |
| Pause | 재생 일시정지 |
| Stop | 재생 중지, playhead를 0으로 리셋 |
| \|< | 이전 키프레임으로 점프 |
| >\| | 다음 키프레임으로 점프 |
| +Key | playhead 위치에 키프레임 추가 |

### 클립 설정

| 항목 | 설명 |
|------|------|
| Time | 현재 playhead 시간 표시 |
| Speed | 재생 속도 배율 (0.1x ~ 5.0x) |
| FR | 프레임 레이트 (1~120 FPS). 키프레임 스냅 그리드 결정 |
| WrapMode | 반복 모드 (Once, Loop, PingPong, Clamp 등) |
| Length | 클립 전체 길이 (초). 키프레임이 범위 초과 시 자동 확장 |
| Save | 변경사항 디스크 저장 (변경 없으면 비활성) |

### 뷰 모드 전환

| 버튼 | 설명 |
|------|------|
| Dope | Dopesheet 모드 — 다이아몬드 마커로 타이밍 편집 |
| Curves | Curves 모드 — Hermite 스플라인 곡선으로 값 편집 |

## 뷰 모드

### Dopesheet

타임라인 위에 트랙별로 키프레임 다이아몬드(◆)를 배치한다. 전체 애니메이션의 타이밍을 한눈에 파악하고, 시간축으로 키프레임을 이동하는 데 최적화되어 있다.

- 노란색 다이아몬드: 선택됨
- 회색 다이아몬드: 선택 안 됨
- 선택한 트랙 행에 파란색 배경 하이라이트

### Curves

프로퍼티 값의 보간 곡선을 직접 시각화한다. 키프레임을 시간·값 2D로 이동할 수 있고, 탄젠트 핸들로 곡선 기울기를 직접 조절한다.

- 트랙별 다른 색상 (6색 순환)
- Y축 자동 피팅 (F키로 리셋)
- 트랙별 가시성 토글 (`v` 버튼)
- 탄젠트 모드: Auto, Linear, Constant, Free

---

## 트랙 관리

### Add Property (+)

트랙 리스트 상단의 `+` 버튼으로 새 프로퍼티 추가:

1. 타겟 Animator의 게임오브젝트 계층 구조가 표시됨
2. 각 오브젝트 하위에 Transform과 컴포넌트 목록
3. Transform → localPosition(x,y,z), localEulerAngles(x,y,z), localScale(x,y,z)
4. 컴포넌트 → public 필드/프로퍼티 (복합 타입은 자동 분해)
   - Vector3 → x, y, z
   - Color → r, g, b, a
5. 이미 추가된 프로퍼티는 비활성화 표시

### Remove Property

트랙을 우클릭 → "Remove Property"로 개별 삭제.

### Remove Group

그룹 헤더를 우클릭 → "Remove Group"으로 그룹 내 모든 프로퍼티 일괄 삭제.

### 트랙 선택

- 클릭: 단일 선택 → 해당 커브만 Curves 뷰에 표시
- Ctrl+클릭: 멀티 선택 → 여러 커브 동시 표시
- Ctrl+A: 전체 선택/해제 토글
- 아무것도 선택하지 않으면 모든 커브 표시

## 이벤트 시스템

룰러 위에 파란색 삼각형 마커로 표시되는 콜백 이벤트.

### 이벤트 추가

룰러 빈 영역을 더블클릭하면 해당 시간에 이벤트 생성.

### 이벤트 Inspector

이벤트를 선택하면 하단에 편집 필드가 표시된다:

| 필드 | 설명 |
|------|------|
| Time | 이벤트 발생 시간 |
| Function | 호출할 콜백 함수 이름 |
| Float | float 파라미터 |
| Int | int 파라미터 |
| Delete | 이벤트 삭제 버튼 |

---

## 프리뷰 시스템

애니메이션 재생/녹화 시, 에디터는 원래 씬 상태를 자동으로 저장(Capture)하고, 중지 시 복원(Restore)한다.

- Play/Record 시작 → 원본 상태 캡처
- Stop 시 → 원본 상태 복원
- PlayMode 진입 시 → 미리보기 강제 종료 + 녹화 해제

---

## Undo / Redo

모든 편집 작업은 Undo 스택에 기록된다:

- 키프레임 추가/삭제/이동/값 편집
- 이벤트 추가/삭제/편집
- 프로퍼티 추가/제거
- 탄젠트 모드 변경
- Record 모드 키프레임 자동 생성
- 클립보드 붙여넣기

Ctrl+Z / Ctrl+Shift+Z 또는 Edit 메뉴에서 실행.

---

## 스냅

모든 키프레임 시간은 프레임 그리드에 자동 스냅된다.

- 그리드 간격 = `1 / frameRate`
- 키프레임 드래그 중 툴팁으로 스냅된 시간 표시
- 프레임 레이트 변경 시 그리드 자동 갱신

## Property Path 규칙

애니메이션 대상 프로퍼티는 문자열 경로로 식별된다:

| 대상 | 경로 예시 |
|------|-----------|
| 자신의 Transform | `localPosition.x` |
| 자신의 컴포넌트 | `SpriteRenderer.color.r` |
| 자식의 Transform | `ChildName.localPosition.x` |
| 자식의 컴포넌트 | `ChildName.SpriteRenderer.color.r` |
