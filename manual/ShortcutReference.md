# IronRose Keyboard Shortcuts

---

## 씬 뷰 (Scene View)

| 입력 | 동작 |
|------|------|
| 우클릭 드래그 | 카메라 회전 (Orbit) |
| 중클릭 드래그 | 카메라 팬 (Pan) |
| 마우스 휠 | 줌 인/아웃 |
| F | 선택 오브젝트에 포커스 |
| 숫자패드 1 | 전면 뷰 (Front) |
| 숫자패드 2 | 후면 뷰 (Back) |
| 숫자패드 3 | 좌측 뷰 (Left) |
| 숫자패드 4 | 우측 뷰 (Right) |
| 숫자패드 5 | 상단 뷰 (Top) |
| 숫자패드 6 | 하단 뷰 (Bottom) |

## 도구 (Tools)

| 단축키 | 도구 |
|--------|------|
| W | Move (이동) |
| E | Rotate (회전) |
| R | Scale (크기) |
| T | Rect (UI 요소) |

## 파일 (File)

| 단축키 | 동작 |
|--------|------|
| Ctrl+N | 새 씬 |
| Ctrl+O | 씬 열기 |
| Ctrl+S | 씬 저장 |
| Ctrl+Shift+S | 다른 이름으로 씬 저장 |

## 편집 (Edit)

| 단축키 | 동작 |
|--------|------|
| Ctrl+Z | 실행 취소 (Undo) |
| Ctrl+Shift+Z | 다시 실행 (Redo) |
| Delete | 선택 오브젝트 삭제 |
| Ctrl+D | 선택 오브젝트 복제 |
| F2 | 이름 변경 |

## 플레이 모드 (Play Mode)

| 단축키 | 동작 |
|--------|------|
| Ctrl+P | 플레이/정지 토글 |
| Play 버튼 | 플레이 모드 시작 |
| Pause 버튼 | 일시 정지 |
| Stop 버튼 | 플레이 모드 종료 |
